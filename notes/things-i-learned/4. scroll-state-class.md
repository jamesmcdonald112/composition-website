# Toggling a class on scroll

A lightweight way to change the navbar appearance when the user scrolls down — no library needed, just a plain scroll event.

```ts
const navbar = document.querySelector<HTMLElement>("[data-navbar]");

if (navbar) {
  const handleScroll = () => {
    if (window.scrollY > 60) {
      navbar.classList.add("is-scrolled");
    } else {
      navbar.classList.remove("is-scrolled");
    }
  };

  window.addEventListener("scroll", handleScroll, { passive: true });
  handleScroll(); // run once on load in case page is already scrolled
}
```

Then in CSS, define what `.is-scrolled` looks like:

```css
.navbar {
  background: transparent;

  &.is-scrolled {
    background: var(--color-bg-brand);
  }
}
```

**Key points:**
- `{ passive: true }` tells the browser this listener won't call `preventDefault()` — allows the browser to scroll without waiting for JS, which improves performance
- Call `handleScroll()` immediately on load so the state is correct if the user lands mid-page or refreshes while scrolled
- The class is defined in CSS, toggled by JS — clean separation
