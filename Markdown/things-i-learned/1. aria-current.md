# aria-current for active nav links

In Astro you can compare the current URL path to each nav link's href and set `aria-current="page"` on the matching link:

```astro
aria-current={currentPath === item.href ? "page" : undefined}
```

Two benefits:
1. **Accessibility** — screen readers announce which link is the current page
2. **CSS hook** — style the active link with `[aria-current="page"]` instead of adding a JS class

```css
.navbar__nav-link[aria-current="page"]::after {
  width: 100%; /* shows the underline on the active link */
}
```

Setting the attribute to `undefined` (instead of `false`) means Astro omits it from the HTML entirely, which is the correct behaviour.
