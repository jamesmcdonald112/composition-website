# Animated underline with ::after

A clean way to animate an underline on hover using a pseudo-element. The line starts at zero width and slides in on hover.

```css
.link {
  position: relative; /* needed so ::after can position itself */

  &::after {
    content: "";
    position: absolute;
    inset-block-end: 0;      /* pins to the bottom of the link */
    inset-inline-start: 0;   /* starts from the left edge */
    width: 0;                /* invisible by default */
    height: 1px;
    background: var(--color-accent);
    transition: width var(--duration-normal) var(--ease-out);
  }

  &:hover {
    color: var(--color-accent);

    &::after {
      width: 100%; /* slides in from left to right */
    }
  }
}
```

Why this is better than `text-decoration: underline`:
- Full control over colour, thickness, position, and animation
- Can be triggered by hover, focus, or a class (e.g. `aria-current="page"`)
- The transition on `width` makes it animate smoothly left to right
